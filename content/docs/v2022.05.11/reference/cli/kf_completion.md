## kf completion

Generate completion script

### Synopsis

To load completions:

Bash:

$ source <(kubectl-kf completion bash)

# To load completions for each session, execute once:
Linux:
  $ kubectl-kf completion bash > /etc/bash_completion.d/kubectl-kf
MacOS:
  $ kubectl-kf completion bash > /usr/local/etc/bash_completion.d/kubectl-kf

Zsh:

# If shell completion is not already enabled in your environment you will need
# to enable it.  You can execute the following once:

$ echo "autoload -U compinit; compinit" >> ~/.zshrc

# To load completions for each session, execute once:
$ kubectl-kf completion zsh > "${fpath[1]}/_kubectl-kf"

# You will need to start a new shell for this setup to take effect.

Fish:

$ kubectl-kf completion fish | source

# To load completions for each session, execute once:
$ kubectl-kf completion fish > ~/.config/fish/completions/kubectl-kf.fish


```
kf completion [bash|zsh|fish|powershell]
```
