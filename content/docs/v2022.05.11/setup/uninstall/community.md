## Using Helm 3

In Helm 3, release names are [scoped to a namespace](https://v3.helm.sh/docs/faq/#release-names-are-now-scoped-to-the-namespace). So, provide the namespace you used to install the operator when installing.

```bash
$ helm uninstall kubeform-provider-*** --namespace kubeform
```

**Note:** For uninstalling Kubeform Module:
```bash
$ helm uninstall kubeform-module --namespace kubeform
```

## Using YAML (with helm 3)

If you prefer to not use Helm, you can generate YAMLs from the Kubeform chart and uninstall using `kubectl`.

```bash
$ helm template kubeform-provider-*** appscode/kubeform-provider-*** --namespace kubeform | kubectl delete -f -
```

**Note:** For uninstalling Kubeform Module:
```bash
$ helm template kubeform-module appscode/kubeform-module --namespace kubeform | kubectl delete -f -
```
